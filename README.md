# FENCE AI

Мобильное приложение-инструмент для компании по строительству заборов и премиум ограждений. Визуализация забора на фото участка и генерация видеооблёта.

## Возможности

- **Визуализация забора** — загрузка/съёмка фото участка, выбор типа забора, превью (интеграция с ИИ — следующий этап).
- **Видеооблёт** — создание видео облёта вокруг участка по визуализации (плейсхолдер под будущий ИИ/API).
- **Проекты** — список проектов с поиском, карточки.
- **Каталог** — типы ограждений (кованый, дерево, металл, сетка, камень, живая изгородь).
- **Ещё** — клиенты, сметы, настройки, о приложении (экран-заглушка под развитие).

## Стек

- React Native, Expo (SDK 54), Expo Router, TypeScript.
- Тестирование: Expo Go (совместимость с Expo Go под SDK 54 на iPhone). Сборка: EAS Build или `expo prebuild` + Android Studio для .apk.

## Запуск и тестирование (Expo Go)

```bash
npm install
npx expo start
```

Откройте проект в Expo Go по QR-коду (Android) или в симуляторе.

**Если Expo Go пишет «Project is incompatible with this version of Expo Go»** — проект переведён на **Expo SDK 54** для совместимости с Expo Go на iPhone. Обновите Expo Go до версии, поддерживающей SDK 54, или используйте этот проект (он уже на 54).

## Сборка .apk (Android)

### Вариант 1: EAS Build (рекомендуется)

1. Установите EAS CLI: `npm i -g eas-cli`
2. Войдите: `eas login`
3. Настройте проект: `eas build:configure`
4. Соберите APK: `eas build -p android --profile preview`  
   В настройках профиля можно указать `"buildType": "apk"` для выдачи именно .apk.

### Вариант 2: Локальная сборка

1. Сгенерируйте нативный проект: `npx expo prebuild`
2. Откройте папку `android` в Android Studio.
3. Build → Build Bundle(s) / APK(s) → Build APK(s).  
   Либо из терминала: `cd android && ./gradlew assembleRelease`  
   APK будет в `android/app/build/outputs/apk/release/`.

## Структура экранов

- **index** — при первом запуске: приветствие, затем редирект в табы.
- **(tabs)** — Проекты | Визуализация | Каталог | Ещё.
- **visualize-upload** — выбор/съёмка фото, выбор типа забора → превью.
- **visualize-preview** — превью с забором, кнопки «Создать видеооблёт» и «Сохранить в проект».
- **visualize-video** — экран генерации и просмотра видеооблёта.

## Дизайн

- Тренды 2026: мягкие скругления, карточки, приглушённая палитра (зелёный, бежевый, тёмно-серый).
- Тема: `constants/Colors.ts`, `constants/Theme.ts`.
- Поддержка светлой и тёмной темы системы.

## Дальнейшая интеграция

- **ИИ-визуализация**: подключить API (например, сегментация + наложение 3D/2D забора или генеративная модель) в экране превью.
- **Видеооблёт**: API генерации видео по изображению (например, NeRF, генеративное видео) в экране `visualize-video`.
- **Бэкенд**: проекты, клиенты, сметы — сохранение и синхронизация с сервером.
